Behavior Implementation

The behavior object
wFORMS.behaviors.behaviorName = { ... }
The behavior's object is:
1. a namespace for all functions and properties related to the behavior.
2. a container for customizable properties.
applyTo() factory method:
1. Inspects the given HTML element and its descendant and determine if and where the behavior should apply.
2. Generates an instance of the behavior
3. Sets event handlers as appropriate to execute the instance's run() method.
wFORMS.behaviors.behaviorName.applyTo = function(element) { }
instance() constructor
var mybehavior = new wFORMS.behaviors.behaviorName.instance(targetElement);
instance is called by the applyTo factory method.
instance.run() method:
Executes the behavior
wFORMS.behaviors.validation.instance.prototype.run = function(e, element) { ... }
Other methods may be implemented as needed.
